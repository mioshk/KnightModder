# ⚔️ KnightModder

**空洞骑士·骑士模组师**


作者：[MioSs-](https://space.bilibili.com/538844794)

*让每一位骑士都能轻松享受 Mod 的乐趣*

---

## 💡 为什么会有这个工具？

很多喜欢《空洞骑士》的朋友，想装 Mod 却总是卡在第一步：

- ❌ 打不开 GitHub，下不动 Mod
- ❌ 不知道去哪里找 Mod 的下载链接
- ❌ 下载了一堆 zip，不知道解压到哪里
- ❌ 装好了 Mod，游戏报错崩溃，不知道哪里出了问题

**KnightModder 就是为了解决这些问题而生的。**

---

## ✨ 它能帮你做什么？

### 🚀 一键搞定，无需折腾

- 选择你的游戏路径，点击安装，剩下的交给工具
- 不需要手动解压、复制、粘贴
- 不需要折腾 API 安装

### 🌐 在线模组仓库

- 所有 Mod 的下载链接集中展示
- 支持一键复制 **本体 + 前置依赖** 的批量下载地址
- 不用再到处找链接、翻帖子

### 🔍 前置依赖检查

- 自动扫描已安装的 Mod，列出缺失的前置依赖
- 不用担心漏装依赖导致游戏崩溃

### 🛠️ Mod 错误检测

- 检测 Mod 安装结构是否正确
- 定位导致游戏报错的问题

---

## 📦 如何使用？

1. **选择游戏路径** → 指向你的 `hollow_knight.exe`
2. **安装 API** → 一键注入 Modding API
3. **浏览在线模组** → 找到想要的 Mod，复制夸克链接
4. **下载 Mod** → 夸克网盘自动跳出转存链接，转存后下载
5. **安装 Mod** → 拖入下载好的.zip或.dll，自动安装
6. **享受游戏** → 启动游戏，Mod 已生效

---


## 📌 注意事项

- 本工具仅支持 **1.5.78 版本** 的《空洞骑士》
- 下载 Mod 需使用 **夸克网盘** App
- 如遇问题，点击首页「检查 Mod 错误」定位故障
