# ⚔️ KnightModder 使用教程

---

## 1. 设置游戏路径

打开软件后，在首页点击 **「浏览」**，选择你的 `hollow_knight.exe` 文件。

---

## 2. 安装 API

点击 **「安装 API」**，为游戏注入 Modding 支持。（仅首次使用需要）

---

## 3. 安装 Mod

1. 切换到 **「在线模组」** 页面
2. 找到想要的 Mod，点击 **「安装」** 或 **「待更新」**
3. 软件自动复制夸克网盘链接到剪贴板
4. 打开夸克网盘，粘贴链接转存并下载
5. 回到软件首页，点击 **「安装 Mod」**，选择刚下载的 `.zip` 或 `.dll` 文件
6. 自动完成安装

---

## 4. 启用/禁用 Mod

切换到 **「本地模组」** 页面，点击 Mod 右侧的开关即可启用或禁用。

---

## 5. 启动游戏

点击首页的 **「立即启动游戏」** 即可。

---

## 6. 检查更新

点击标题栏的 **「检查更新」** 按钮，查看软件是否有新版本。
